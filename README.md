# Agent Mission Control

Milestone 0 scaffold for the Agent Mission Control UI.

## Milestone 0 Scope

- Next.js app scaffolded with TypeScript and App Router.
- shadcn/ui initialized and wired to Tailwind v4.
- Darkmatter theme tokens retained in `app/globals.css`.
- Core shell implemented with sidebar navigation, top action bar, and dashboard placeholder cards/sections.

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS v4
- shadcn/ui
- Lucide icons

## Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Scripts

- `npm run dev` - Start local development server.
- `npm run lint` - Run ESLint checks.
- `npm run build` - Create production build.
- `npm run start` - Run production server.
